# Glossary

Status: Milestone 1 seed - 2026-07-17

This alphabetical index defines terms used by the first three drafted concepts. It will grow with the guide. Definitions stay short and link to the canonical explanation rather than duplicating it.

## B

**Base (or radix)**  
The number of digit values used by a positional number system: decimal is base 10, [binary](chapters/01-computing-foundations.md#binary-numbers) is base 2, octal is base 8 and hexadecimal is base 16.

**Binary**  
A base-2 positional number system using only `0` and `1`; see [Binary numbers](chapters/01-computing-foundations.md#binary-numbers).

**Bit**  
A binary digit with one of two values, `0` or `1`; see [Bits and bytes](chapters/01-computing-foundations.md#bits-and-bytes).

**Byte**  
A group of eight bits; see [Bits and bytes](chapters/01-computing-foundations.md#bits-and-bytes).

## E

**Encoding**  
A defined way of mapping information to another representation, such as mapping Unicode values to bytes with UTF-8.

## H

**Hexadecimal (hex)**  
A base-16 positional notation using digits `0` to `9` and letters `A` to `F`; see [Hexadecimal and octal](chapters/01-computing-foundations.md#hexadecimal-and-octal).

## O

**Octal**  
A base-8 positional notation using digits `0` to `7`; see [Hexadecimal and octal](chapters/01-computing-foundations.md#hexadecimal-and-octal).

**Octet**  
An unambiguous name for an eight-bit byte, commonly used in Internet standards; see [Bits and bytes](chapters/01-computing-foundations.md#bits-and-bytes).

## P

**Positional number system**  
A way of writing numbers in which a digit's value depends on its position, as explained in [Binary numbers](chapters/01-computing-foundations.md#binary-numbers).

## U

**Unicode**  
A standard that assigns numbers to the characters and other text elements used by writing systems; an encoding such as UTF-8 represents those values as bytes.

**Unsigned integer**  
A whole-number representation with no negative values; an unsigned byte can represent values from 0 through 255.

**UTF-8**  
A variable-length Unicode encoding that represents a Unicode scalar value with one to four bytes.
