# Triage labels

The engineering skills use five canonical triage roles. The public GitHub repository uses the same strings.

| Role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | A maintainer must evaluate the issue. |
| `needs-info` | `needs-info` | More information is required from the reporter. |
| `ready-for-agent` | `ready-for-agent` | The issue is fully specified and can be handled without additional context. |
| `ready-for-human` | `ready-for-human` | Human judgement or implementation is required. |
| `wontfix` | `wontfix` | The issue will not be actioned. |

When a skill names a role, apply the corresponding GitHub label from this table.
